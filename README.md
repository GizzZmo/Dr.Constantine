# Dr. Constantine — Loopens Tyranni og Frihet

En interaktiv doktoravhandling i moderne musikkteori av **Jon-Arve Constantine**.

## Om prosjektet

`index.html` er et selvinneholdt, interaktivt web-dokument som utforsker psykoakustikken, matematikken og teknologien bak den moderne musikkloopen. Avhandlingen argumenterer for at spenning og frigjøring i dagens musikk ikke primært oppnås gjennom tradisjonell akkordprogresjon, men via frekvensmanipulasjon, teksturendringer og lyriske ledemotiver.

## Teknologi

- Ren HTML/CSS/JavaScript — ingen byggsteg nødvendig
- [Tailwind CSS](https://tailwindcss.com/) (CDN) for layout og styling
- [Chart.js](https://www.chartjs.org/) (CDN) for interaktive grafer
- Google Fonts: *Share Tech Mono* og *Inter*
- SoundCloud Embedded Player for empirisk lydarkiv

## Bruk

Åpne `index.html` direkte i en nettleser. Ingen server eller installasjon kreves.
